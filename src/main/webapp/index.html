Game API Usage Guide

This guide provides information on how to use the Game API. You can perform operations such as retrieving, creating, updating, and searching for games.

API Endpoints

Get All Games

Retrieve a list of all games.

GET /api/games

Get Game by ID

Retrieve details for a specific game based on its ID.

GET /api/games/{id}

Create New Game

Create a new game with the specified details.

POST /api/games

Example JSON request:

{
    "title": "New Game",
    "developer": "Game Developer",
    "description": "Description of the game",
    "release_date": "2024-01-01",
    "upc": "123456789012"
}

Update Game

Update the details of an existing game.

PATCH /api/games/{id}

Example JSON request:

{
    "title": "Updated Game Title",
    "developer": "Updated Developer",
    "description": "Updated description",
    "release_date": "2024-01-01",
    "upc": "123456789012"
}

Find Games by Developer

Search for games based on the developer's name.

GET /api/games/developer/{developer}

Find Games by Title

Search for games based on the title.

GET /api/games/title/{title}
